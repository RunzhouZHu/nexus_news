Node Action Buttons Placement

When a node is selected, where and how do the action buttons appear?

A — Radial Halo

Action icons orbit the node in a halo ring on click. Pin and Load More appear as icon buttons at fixed angles around the circle. Smooth, keeps focus on the node.

B — Side Pill Bar

A compact pill-shaped button bar appears to the right of the node. Stacked vertically: 📌 Pin, ➕ Load More. Clear, easy to click, minimal overlap with the graph.

C — Floating Strip (above node)

A horizontal button strip floats just above the node: [📌 Pin] [➕ More]. Like a mini toolbar that follows the node. Familiar, similar to Figma/Miro object toolbars.
