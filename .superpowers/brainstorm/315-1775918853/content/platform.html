What platform is this built for?

A zoomable knowledge graph works very differently on a 6" phone screen vs a desktop browser. This choice affects everything.

A

Web App (Desktop-first)

Built for browsers on laptop/desktop. The graph has room to breathe — pan, zoom, hover previews, keyboard shortcuts. Best experience for exploring complex networks of events. Shareable via URL.

B

Mobile App (iOS / Android)

Touch-native. Pinch to zoom the graph, swipe to navigate. Push notifications for breaking news connections. Reaches the widest general audience. Graph needs to be carefully simplified for small screens.

C

Both — Web + Mobile

Responsive web app that works on all screen sizes, plus a dedicated mobile app later. Start with web to validate the concept, then expand. More scope upfront but broadest reach.
